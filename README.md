## Flagship Project

### 🚨 AEGIS — Real-Time Emergency Response Platform

A full-stack emergency coordination system: five Spring Boot microservices, three FastAPI AI services, and a real-time command center frontend — built to route ambulances, predict severity, and recommend hospitals under live load.

| Metric | Result |
|---|---|
| P99 Latency | **87ms** |
| Concurrent Users Supported | **1,000+** |
| Query Load Reduction | **50%** |

**Stack:** Java · Spring Boot · Python · FastAPI · PostgreSQL · WebSockets/STOMP · React/Next.js · Docker · Prometheus/Grafana

[**Live Demo**](https://lifelineai-git-main-alsas-projects-737cc79e.vercel.app) · [**Repository**](https://github.com/Ansh811-hub/AEGIS)
